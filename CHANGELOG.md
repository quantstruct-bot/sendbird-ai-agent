## New features

### iOS sample app (QuickStart)

We’ve added a brand new iOS sample app, “QuickStart,” to help you quickly integrate and test Sendbird AI agent capabilities in your own iOS projects. This sample demonstrates how to initialize the AI agent SDK, manage user sessions, handle push notifications, and present AI-driven chat conversations in a modern Swift UI. You’ll also find example implementations for customizing agent themes, managing context, and toggling between light and dark modes.

The QuickStart app includes ready-to-use code for:
- Connecting and disconnecting from the Sendbird AI agent service
- Handling push notifications and deep linking into AI-powered conversations
- Presenting the agent chat interface and launcher button
- Customizing look and feel with theme switching

Explore the sample to accelerate your own iOS AI agent integration and deliver seamless, intelligent chat experiences for your users.

## Improvements
